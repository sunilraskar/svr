Getting Started with Github - Step by Step Guide for bignners

Be a great team player - manage your soruce code using the github.

Posted by Sunil on Feb 13, 2018

Step 1:

Download and install Git.

Windows user download git for Windows from HERE

Linux user download git for Linux from HERE

Step 2:

create a new repository

Open the DOS prompt and create a new directory at your desired location for example C:\MyFristRepo

 git init  

This will create a new git local repository

In case you do not have a repository in your github account, You will to create a new public repository by click on the New Repository button. Note the URL for example https://github.comusername/MyFristRepo.git

Step 3:

Add Soruce Codes or Projects folders

aaaa

Copy a small size project soruce code and releated folders and sub folders to C:\MyFirstRepo

Add all folders, subfolders and the files to local git source control

 add .  

Checkin or cmmit the newly added files in the local reposiotry

  git commit -m "inital commit"  

connect your local respository folder to remote server respository

 git remote add origin https://github.com/username/MyFristRepo.git  

Push the folders, subfolders and files to remote server

  git push origin master
